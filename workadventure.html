Guía de Uso para Reuniones en WorkAdventure

Cómo Unirse a una Reunión

Para iniciar una videollamada, simplemente acércate a otros avatares. La llamada comenzará automáticamente cuando estés dentro de su burbuja de proximidad.

Compartir Contenido

Presiona el botón de "Compartir Pantalla" en la barra de herramientas inferior cuando necesites mostrar documentos o presentaciones.

Uso de Salas Privadas

Entra a una sala de reuniones cerrando la puerta detrás de ti para mantener la privacidad. Las salas tienen capacidad limitada para mejor experiencia.

Herramientas Colaborativas

Presiona la tecla 'E' para interactuar con pizarras y otros objetos. Múltiples usuarios pueden colaborar simultáneamente.

Protocolo Durante las Reuniones:

  • 🎤 Silencia tu micrófono cuando no estés hablando
  • ✋ Usa el emoji de "mano levantada" para pedir la palabra
  • 💬 Utiliza el chat para preguntas sin interrumpir
  • 🚶‍♂️ Mantén tu avatar a una distancia apropiada de otros participantes
  • ⏰ Respeta los horarios establecidos para las reuniones
